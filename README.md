# 🌤️ Weather Application

> A clean and responsive weather app built with React, TypeScript, and Tailwind CSS — search any city and get live weather data instantly.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Live temperature, humidity & conditions
- 💨 Wind speed and weather description
- 📱 Fully responsive — works on mobile & desktop
- ⚡ Fast & lightweight with Vite + Bun

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/Abhinikesh/Weather_Application.git
cd Weather_Application
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run locally
```bash
npm run dev
```

> Open `http://localhost:5173` in your browser.
